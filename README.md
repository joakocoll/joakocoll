<h2 align="center">🚀 Featured Projects</h2>

### 🏛️ [ClubsCuotas](https://github.com/joakocoll/clubCluotas)
Full-stack system for managing sports club members and membership fees.
Containerized microservices architecture with isolated build contexts in a monorepo.

```
clubsCuotas/
├── backend/      # RESTful API + SQLAlchemy ORM
├── frontend/     # Svelte SPA + Vite internal proxy
├── secrets/      # Credentials (gitignored)
└── docker-compose.yml
```

`Python 3.12` · `FastAPI` · `SQLAlchemy` · `Svelte + Vite` · `MariaDB 10.11` · `Docker Compose`

---

### 🎙️ [Whisper Audio CLI](https://github.com/joakocoll/whisper-cli)
CLI to transcribe audio files and search for keyword or phrase matches using Whisper + Fuzzy matching.
Supports `.wav` · `.mp3` · `.m4a` · `.ogg` — optional GPU via CUDA/PyTorch.

`Python 3.9+` · `OpenAI Whisper` · `thefuzz` · `ffmpeg` · `CUDA (optional)` · `MIT License`

---

### 🖥️ Home Lab Server
Self-hosted stack with KVM, Docker Compose, multi-layer backups (Rclone + ZSTD), network monitoring bot via Telegram API and graceful shutdown on power failure.

`Ubuntu 24.04` · `KVM/Libvirt` · `Docker` · `Bash` · `Systemd`
